# TP5 - FastAPI with PostgreSQL

## Concepts Covered

- FastAPI with SQLAlchemy
- PostgreSQL Database Connection
- ORM Models
- Pydantic Models
- Dependency Injection
- CRUD API Endpoints

## Libraries Used

- fastapi
- sqlalchemy
- psycopg2-binary
- uvicorn

## How to Run

uvicorn main:app --reload

Open:
http://127.0.0.1:8000/docs

## Conclusion

This project connects FastAPI with PostgreSQL using SQLAlchemy ORM.
It demonstrates how to build a real backend API with a relational database.
